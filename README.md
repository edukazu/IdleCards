# Idle Cards

Jogo mobile idle de cartas em desenvolvimento.

## Tecnologia

- Unity 6.5 (6000.5.1f1)
- C#
- Visual Studio Code
- Android como plataforma inicial
- Build Web da Unity para revisões rápidas
- Git e GitHub

## Direção atual

- Jogo em orientação vertical.
- Inimigos de PvE na parte superior da tela.
- Quatro cartas do jogador na parte inferior.
- Formação em cruz: norte, sul, leste e oeste.
- Qualquer carta poderá ocupar qualquer posição.
- Buffs, debuffs e prioridade de alvo por slot ficam para uma fase futura.

Consulte `Docs/PROJECT_STATE.md` antes de continuar o desenvolvimento.
