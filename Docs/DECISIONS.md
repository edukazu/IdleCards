# Registro de Decisões

## DEC-001 — Programador principal

GPT-5.5 Thinking será responsável pela arquitetura e pelo código principal.

## DEC-002 — Engine

Usar Unity 6.5, versão 6000.5.1f1.

## DEC-003 — Plataforma

Priorizar Android e usar build Web da Unity para revisões.

## DEC-004 — Metodologia

Desenvolver por pequenas entregas e aguardar aprovação antes de avançar.

## DEC-005 — Tela de batalha

Usar orientação vertical, PvE na parte superior e quatro cartas do jogador em cruz na parte inferior.

## DEC-006 — Slots

Qualquer carta poderá ocupar qualquer posição. Efeitos próprios dos slots serão adicionados futuramente.
