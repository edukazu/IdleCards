# Continuidade em Outro Chat

Estamos desenvolvendo o projeto Idle Cards.

## Papéis

- Eduardo dirige, orienta, testa e aprova.
- GPT-5.5 Thinking atua como programador principal e arquiteto técnico.

## Tecnologia

- Unity 6.5, versão 6000.5.1f1.
- C#.
- Visual Studio Code.
- Android.
- Build Web da Unity para revisão.
- Git e GitHub.

## Método

Trabalhar em pequenas entregas. Não avançar várias etapas sem a confirmação de Eduardo.

## Tela de batalha aprovada

- Orientação vertical.
- PvE na parte superior.
- Quatro cartas do jogador na parte inferior.
- Formação em cruz: norte, sul, leste e oeste.
- Qualquer carta pode ocupar qualquer slot.
- Buffs, debuffs e prioridade de alvo por slot ficam para depois.

## Estado atual

O Editor Unity abriu e os módulos Android e Web estão instalados. Ainda não há gameplay implementado.
